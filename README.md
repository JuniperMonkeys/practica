# practica
"Simple Jira for Real Life"
Practica is a fork of a tool I'd made for shared travel management called <a href="https://cloudhat.org">Cloud Hat</a>. The main addition is per-task user assignment; the intention is to make it a good tool for shared workplace event management. Atlassian is happy to provide a way more powerful way(s) to do this, but the idea was to make "kanban for people who don't care what that means".
